### Verificar a versão instalado do Node no seu computador.
- node -v


### Verificar a versão instalado do NPM no seu computador.
- npm -v


### Comandos para criar o projeto.
- npm create vite@latest


### Iniciar o NPM
- npm install


### Rodar o projeto.
- npm run dev
